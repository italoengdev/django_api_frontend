This is a sample application built on Next.js and Tailwind CSS that allows users to perform CRUD operations on an API built on Django. The application is deployed on Vercel at https://django-api-frontend.vercel.app/.

## Features
The application provides the following features:

- View a list of objects retrieved from the Django API
- Create a new object by filling out a form
- Update an existing object by editing its fields in a form
- Delete an existing object

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
